# Machine_Learning_-_Projeto_de_Credit_Score_Naive_Bayes_-_EBAC
Projeto da EBAC que aplica o algoritmo Naive Bayes para prever o score de crédito (baixo, médio e alto). Inclui validação das bases, treinamento e avaliação do modelo por meio de acurácia, recall e matriz de confusão, alcançando alto desempenho e boa capacidade de generalização.
